## Code Analysis Findings

### 1. Atom Definition Patterns

**State Atoms (Mutable):**
```swift
// Pattern: KeyValue store with defaultValue
struct CounterAtom: SKStateAtom, Hashable {
    typealias Value = Int  // 📌 Generated by @StateAtom
    func defaultValue(context: SKAtomTransactionContext) -> Int { 0 }
}
// Boilerplate: typealias, Hashable conformance, module conformance
```

**Value Atoms (Derived):**
```swift
// Pattern: Watch other atoms + compute
struct DoubledCounterAtom: SKValueAtom, Hashable {
    typealias Value = Int  // 📌 Generated by @ValueAtom
    func value(context: SKAtomTransactionContext) -> Int {
        context.watch(CounterAtom()) * 2  // Manual dependency tracking
    }
}
```

**Task Atoms (Async):**
```swift
// Pattern: Async fetch with automatic phase tracking
struct FetchAtom: SKTaskAtom, Hashable {
    typealias TaskSuccess = String  // 📌 Generated by @TaskAtom
    func task(context: SKAtomTransactionContext) async -> String {
        let id = context.watch(RequestSeedAtom())  // Auto-parameter
        return "Data for \(id)"
    }
}
```

### 2. Factory Function Patterns (Inline Atoms)

```swift
// ✅ Already minimal - using factory functions

let inlineSummaryAtom = selector { context in ... }
let memberScoreAtom = atomFamily { (id: Int) in id * 10 }

```

## 🎯 Recommended Macros (Priority Order)

### HIGH PRIORITY (High ROI, Common Patterns)

#### 1. **@AtomFamily** — Parameterized State Atoms
```swift
// Before: Manual factory function
let userAtom = atomFamily { (id: String) in
    UserState(id: id)  // Must manually structure
}

// After: @AtomFamily macro
@AtomFamily
struct UserAtom {
    let userID: String
    typealias Value = User
    
    func defaultValue(context: SKAtomTransactionContext) -> User {
        User(id: userID)
    }
}
// Generates: public let userAtom = atomFamily { (userID: String) in ... }
```

**Boilerplate Saved:** Generates factory function + proper ID parameter extraction  
**Usage Count:** 4+ examples in codebase with atomFamily  
**Recommendation:** ⭐⭐⭐ Implement

---

#### 2. **@SelectorFamily** — Parameterized Derived Atoms
```swift
// Before: Manual selectorFamily with context parameter
let memberLabelAtom = selectorFamily { (id: Int, context: SKAtomTransactionContext) in
    let score = context.watch(memberScoreAtom(id))
    return "Member \(id)"
}

// After: @SelectorFamily macro
@SelectorFamily
struct MemberLabelAtom {
    let memberID: Int
    typealias Value = String
    
    func value(context: SKAtomTransactionContext) -> String {
        let score = context.watch(memberScoreAtom(memberID))
        return "Member \(memberID)"
    }
}
// Generates: public let memberLabelAtom = selectorFamily { (memberID: Int, ctx) in ... }
```

**Boilerplate Saved:** Factory wrapper + parameter handling  
**Usage Count:** 2+ examples (memberLabelAtom pattern)  
**Recommendation:** ⭐⭐⭐ Implement

#### 4. **@AtomReducer** — State Atoms with Action Dispatch
```swift
// Similar to @HookReducer but generates Atom + store actions
@AtomReducer
struct CounterAtom {
    typealias State = Int
    enum Action { case increment, decrement }
    
    func reduce(_ state: inout Int, action: Action) {
        switch action {
        case .increment: state += 1
        case .decrement: state -= 1
        }
    }
}

// Generates:
// struct CounterAtom: SKStateAtom { ... dispatch(Action) ... }
// Lets you: atomContext.dispatch(CounterAtom(), action: .increment)
```

**Boilerplate Saved:** Eliminates manual action threading  
**Pattern Frequency:** 3+ useReducer examples in codebase  
**Recommendation:** ⭐⭐⭐ Implement

**Boilerplate Saved:** Effect registration helpers  
**Current Usage:** No current examples, but powerful pattern  
**Recommendation:** ⭐⭐ Consider

#### 6. **@ComputedAtom** — Shorthand for Simple Selectors
```swift
// Before: Full ValueAtom struct
struct DoubledCounterAtom: SKValueAtom, Hashable {
    typealias Value = Int
    func value(context: SKAtomTransactionContext) -> Int {
        context.watch(CounterAtom()) * 2
    }
}

// After: Inline computed atom
@ComputedAtom(watching: CounterAtom)
func doubledCounterAtom(context: SKAtomTransactionContext) -> Int {
    context.watch(CounterAtom()) * 2
}
```

**Boilerplate Saved:** 5 lines → 2 lines  
**Current Usage:** 2 examples (DoubledCounterAtom, FormattedAtom)  
**Recommendation:** ⭐⭐ Nice to have
